# Storyteller Android SDK Main Sample

This is a Jetpack Compose based sample. The Storyteller SDK is installed via Gradle.

## Prerequisites

- Android Studio 2022.3.1

## Getting Started

1. Open the project in Android Studio
2. Run the project
